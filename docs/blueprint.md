# **App Name**: CareerCompass VA

## Core Features:

- Questionnaire: 15 question assessment, each answer weighted across Administrative, Creative, Technical, Analytical, and Customer-Centric categories.
- Result Profile Calculation: Calculate a percentage score for each category based on user's answers.
- Career Path Recommendation: Recommend a VA career path based on the category with the highest score.
- Roadmap Generation: Generate a 4-step roadmap (Learn, Practice, Portfolio, Apply) based on the recommended career path using generative AI tool.
- Data Storage: Saves the user's result profile and timestamp to Firestore.
- Data Visualization: Display results with progress bars.
- Firestore Persistence: Persist all result profiles to a Firestore collection.

## Style Guidelines:

- Primary color: Soft blue (#A0C4FF) for trust and professionalism.
- Background color: Very light desaturated blue (#F0F8FF).
- Accent color: Light purple (#BDB2FF) for highlighting and calls to action.
- Body and headline font: 'Inter' (sans-serif) for a modern, neutral, and objective feel.  
- Use clean, professional icons related to VA skills (e.g., calendar for admin, pen for creative).
- Employ a modern and empathetic design with cards for questions.
- Use subtle animations on the roadmap timeline and transitions for a smooth user experience.
